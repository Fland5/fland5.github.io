ElastAlert官网链接

环境：centos7.0-64位

PIP（PIP是Python包管理工具，该工具提供了对Python 包的查找、下载、安装、卸载的功能。）安装

  1. $ curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py # 下载安装脚本
  2. $ sudo python get-pip.py
  3. pip --version

    ElastAlert目前不支持python 3.x，因此这里安装的pip对应python版本为2.7。

    ElastAlert安装

  4. git clone https://github.com/Yelp/elastalert.git
  5. 若无git，安装命令如下：
    yum install git

  6. 再次执行第1步
  7. 安装模块
    pip install "setuptools>=11.3"
  8. 安装依赖
    非常重要的一步！！！
    yum -y install glib gcc python-devel libffi-devel openssl-devel
  9. 安装ElastAlert
    cd /usr/elastalert
    python setup.py install
  10. 安装elasticsearch-py的适合版本
    pip install "elasticsearch>=5.0.0"

    ElasticStack本人选用的是5.6.15版本。

    ElastAlert配置

    config.yaml配置

  12. vim config.yaml
    若无vim，安装命令如下：
    yum install vim
    配置文件内容格式中的空格、段落一定要注意保持！！！

ElastAlert运行

  1. 三条命令
    ElastAlert安装好后会多出如下三条命令：
    1. elastalert-create-index
      这个命令会在Elasticsearch创建默认名为elastalert_status的索引，用于将ElastAlert有关查询及其警报信息和元数据保存至Elasticsearch，推荐创建。

curl '10.10.61.205:9200/_cat/indices?v'
查询Elasticsearch索引。

  1. elastalert-test-rule
    测试自定义配置规则是否正确。

    elastalert-test-rule ./example_rules/rule.yaml

    异常：yaml.parser.ParserError: expected ‘‘, but found ‘‘…
    解决：
    注意name前的两个英文空格。
  2. elastalert-rule-from-kibana
    从 Kibana3 已保存的仪表盘中读取 Filtering 设置，帮助生成 config.yaml 里的配置。不过注意，它只会读取filtering，不包括queries。
    1. ElastAlert启动
      nohup python -m elastalert.elastalert --rule rule.yaml --verbose &
